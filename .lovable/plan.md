
# Flowビューのレイアウト変更

## 概要
Flowビューを X (Twitter) 風のシンプルな縦スクロールレイアウトに変更します。左サイドバーを廃止し、入力フォームとログ一覧を縦に並べ、日付ナビゲーションをヘッダーに移動します。

## 日付ナビゲーション仕様

### コンポーネント構成

| ボタン | 動作 | 表示条件 |
|--------|------|----------|
| ◀（前日） | 前日に移動 | 常に表示 |
| 日付表示 | 現在選択中の日付（例：2月1日(土)） | 常に表示 |
| ▶（次日） | 次日に移動 | 今日でなければ表示（未来は不可） |
| 今日 | 今日に戻る | 今日でなければ表示 |
| 📅（カレンダー） | カレンダーポップオーバーを開く | 常に表示 |

### ナビゲーションのUI
```typescript
// ヘッダー内のナビゲーション部分
<div className="flex items-center gap-1">
  <Button variant="ghost" size="icon" onClick={goToPrevDay}>
    <ChevronLeft className="h-4 w-4" />
  </Button>
  
  <span className="text-sm font-medium min-w-[100px] text-center">
    {isToday ? '今日' : '2月1日(土)'}
  </span>
  
  {!isToday && (
    <Button variant="ghost" size="icon" onClick={goToNextDay}>
      <ChevronRight className="h-4 w-4" />
    </Button>
  )}
  
  {!isToday && (
    <Button variant="ghost" size="sm" onClick={goToToday}>
      今日
    </Button>
  )}
  
  <Popover>
    <PopoverTrigger asChild>
      <Button variant="ghost" size="icon">
        <Calendar className="h-4 w-4" />
      </Button>
    </PopoverTrigger>
    <PopoverContent>
      <Calendar ... />
    </PopoverContent>
  </Popover>
</div>
```

## 削除する要素

| 要素 | ファイル | 理由 |
|------|----------|------|
| 左サイドバー | Dashboard.tsx | 新ナビゲーションで代替 |
| モバイルのサイドバーシート | Dashboard.tsx | 不要に |
| FlowView内のヘッダー | FlowView.tsx | Dashboard.tsxのヘッダーに統合 |
| 今日に戻るボタン | FlowView.tsx | ヘッダーナビゲーションに移動 |

## 変更対象ファイル

| ファイル | 変更内容 |
|----------|----------|
| `src/pages/Dashboard.tsx` | ヘッダーに日付ナビを追加、サイドバー削除、レイアウト簡略化 |
| `src/components/flow/FlowView.tsx` | ヘッダーセクション削除（日付表示はDashboardに移動）、シンプル化 |

## 詳細な変更内容

### Dashboard.tsx
1. **サイドバー関連を完全削除**
   - `Sheet` コンポーネント（モバイルメニュー）削除
   - `aside`（デスクトップサイドバー）削除
   - `sidebarOpen` state 削除

2. **ヘッダーに日付ナビゲーションを追加**
   - 新規作成: `DateNavigation` コンポーネント（インラインまたは別ファイル）
   - ロゴの右側に配置
   - 前日/次日/今日/カレンダーボタン

3. **レイアウトの簡略化**
   - `TabsContent value="flow"` の中身をシンプルに FlowView のみに

### FlowView.tsx
1. **ヘッダーセクション削除**
   - 380-419行目の「Header」div を削除
   - 「今日に戻る」ボタンはDashboardのヘッダーに移動済み

2. **セクションヘッダーの調整**
   - 現在の「今日のログ / この日のログ」タイトルは維持

## 技術的考慮事項

### 過去日表示時のUI
- FlowInputは `isToday` を受け取り、過去日では背景色が変わる（既存動作を維持）
- 日付ナビゲーションで現在日が一目でわかるようにする

### カレンダーポップオーバー
- 既存の `DateSelector` の Calendar 部分を再利用
- `datesWithEntries` を渡してエントリのある日にドットを表示

### レスポンシブ対応
- モバイルでは日付テキストを短縮（「2/1」形式）
- ボタンサイズはモバイルでタップしやすいサイズを維持
