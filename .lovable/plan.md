
# アナリティクス画面にストリーク表示を追加

## 概要
`entries` テーブルの日付データから連続投稿日数（ストリーク）を計算し、アナリティクス画面の上部にカード形式で表示する。

## 変更対象

### `src/hooks/useAnalytics.ts`
- `entries` テーブルから全日付を取得するクエリを追加
- JST タイムゾーンで今日から遡り、連続投稿日数を計算するロジックを実装
- 返却値に `currentStreak`（現在の連続日数）と `longestStreak`（最長記録）を追加

### `src/pages/Analytics.tsx`
- 期間セレクターの下、グラフの上にストリークカードを追加
- 表示内容:
  - 現在のストリーク: 「🔥 N日連続投稿中」（今日も投稿済みの場合）または「⚡ 昨日まで N日連続！」（今日未投稿だが延長可能な場合）
  - 最長ストリーク: 「🏆 最長記録: N日」
  - 投稿がない場合: 「✨ 今日から始めよう」

## ストリーク計算ロジック
1. entries テーブルから `date` カラムのユニークな日付一覧を取得
2. JST の今日の日付から1日ずつ遡り、投稿がある日をカウント
3. 今日に投稿があれば今日からカウント開始、なければ昨日からカウント開始
4. 最長ストリークは全日付を走査して計算

## UI イメージ

```text
┌─────────────────────────────────┐
│ 🔥 12日連続投稿中   🏆 最長: 25日 │
└─────────────────────────────────┘
```

ストリークは期間フィルター（7/30/90日）に依存せず、常に全期間で計算する。
